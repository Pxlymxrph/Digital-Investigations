# Digital Investigations

Repositorio que reúne investigaciones técnicas, análisis forenses, casos OSINT, análisis de amenazas y estudios de seguridad realizados con fines educativos, profesionales y de desarrollo de habilidades en ciberseguridad.

## Áreas Cubiertas

### Mobile Forensics

* Análisis de dispositivos móviles.
* Identificación de hardware.
* Validación de autenticidad.
* Análisis de artefactos.

### OSINT

* Investigaciones de fuentes abiertas.
* Identificación de perfiles fraudulentos.
* Correlación de información pública.
* Análisis de presencia digital.

### Phishing & Threat Analysis

* Investigación de enlaces maliciosos.
* Análisis de infraestructura.
* Recolección de indicadores de compromiso.

### Web Security

* Análisis de aplicaciones web vulnerables.
* Identificación de vulnerabilidades.
* Documentación técnica.

### Malware Analysis

* Análisis estático y dinámico.
* Identificación de comportamiento.
* Documentación de hallazgos.

## Objetivo

Documentar investigaciones reales y laboratorios prácticos para fortalecer habilidades en:

* Ciberseguridad
* OSINT
* DFIR
* Threat Intelligence
* Pentesting
* Investigación Digital
