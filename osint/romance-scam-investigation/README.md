\# OSINT Case Study 01

\## Investigación de posible scam romántico y sextorsión



Investigación OSINT realizada sobre un perfil conocido mediante una aplicación de citas que presentaba inconsistencias de identidad, ubicación y comportamiento.



\---



\## Objetivo



Verificar la autenticidad del perfil, identificar inconsistencias en su narrativa y reducir el riesgo para la persona afectada.



\---

\## Hallazgos Principales



| Hallazgo | Confianza |

|-----------|-----------|

| Inconsistencia geográfica | Alta |

| Negativa persistente a videollamadas | Alta |

| Narrativa contradictoria | Alta |

| Love Bombing | Alta |

| Riesgo de sextorsión | Muy Alta |

| Material visual posiblemente reutilizado | Alta |

\## Resultado



La investigación permitió identificar múltiples indicadores compatibles con fraude romántico y posible sextorsión.



Las medidas de contención implementadas permitieron reducir el riesgo para la persona afectada y preservar evidencia relevante.
